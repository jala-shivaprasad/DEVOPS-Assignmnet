🐾 About Animal Classifier

This app uses a DenseNet121 model fine-tuned on the Animals-10 dataset to classify images into 10 animal categories.

Grad-CAM (Gradient-weighted Class Activation Mapping) is used to visualise which regions of the image the model paid most attention to when making its prediction.

The model was trained in TensorFlow/Keras using transfer learning. Input images are resized to 128×128 pixels before inference.
